# 💰 Экономический Telegram-бот "Bomzh"

Многопользовательская экономическая игра с реалистичной системой работы, бизнеса, аукционов и казино.

## ✨ Основные возможности
*   **Работа:** Выполняйте мини-игры на разных работах (таксист, программист, пилот и др.).
*   **Бизнес:** Покупайте и улучшайте бизнес, получайте пассивный доход.
*   **Экономика:** Развитая система баланса, переводов и реферальная программа.
*   **Казино:** Игры в кости, блэкджек и "Колесо Фортуны" с настраиваемыми ставками.
*   **Аукцион:** Продавайте свой бизнес другим игрокам.
*   **Администрирование:** Полный набор команд для управления игрой.

## 🛠 Установка и запуск

1.  **Установите зависимости и проект:**
    ```bash
    pkg install git python3
    pip install aiogram aiosqlite
    git clone https://github.com/NVMR-bruh/bomzh-bot-0.1.git
    cd bomzh-bot-0.1
    ```
2.  **Настройте бота:**
    *   В файле `bot.py` найдите строки `BOT_TOKEN` и `ADMIN_ID` и замените значения на свои.
3.  **Запустите бота:**
    ```bash
    python3 bot.py
    ```

## 📚 Команды бота
*   `/start` - Регистрация в игре.
*   `/admin` - Панель администратора (только для `ADMIN_ID`).
